## Uses

Software I use and tools I love.

---

### Hardware

- [MacBook Air M2](https://www.apple.com/macbook-air/) *— Thin, light, and powerful enough for everyday developer workflows. Silent and lasts all day.*

---

### Editor + Terminal

**Editor**

- [Zed](https://zed.dev/) *— Fast, collaborative code editor built in Rust. Clean UI and native performance.*
- [IntelliJ IDEA](https://www.jetbrains.com/idea/) *— The heavyweight champion for Java and complex projects. Worth every MB of RAM.*
- [Antigravity CLI](https://antigravity.dev/) *— Context-aware autonomous AI coding agent.*

**Terminal**

- [Alacritty](https://alacritty.org/) *— GPU-accelerated terminal emulator. Blazingly fast and minimal.*

---

### Productivity

- [Raycast](https://www.raycast.com/) *— Command center for macOS. Launcher, calculator, clipboard manager, and everything in between.*
- [Dia Browser](https://diabrowser.com/) *— Privacy-focused browser by The Browser Company. Clean, fast, and organized.*
- [Obsidian](https://obsidian.md/) *— Note-taking and personal wiki management.*
- [ChatWise](https://chatwise.app/) *— Clean LLM interface for ChatGPT, Claude, and other models. No bloat, just conversations.*

---

### Sound & Ambience

- [FluidVoice](https://fluidvoice.io/) *— Voice modulation and audio processing for calls and recordings.*
- [Thock](https://thock.app/) *— Mechanical keyboard sound effects. Because typing should sound satisfying.*

---

### Development Tools

- [GitHub Desktop](https://github.com/apps/desktop) *— Simple repository management without the terminal gymnastics.*
- [Postman](https://www.postman.com/) *— API testing and development platform.*

---

### Design & Media

- [Figma](https://www.figma.com/) *— Design tool for interfaces and prototypes. Even non-designers can make things look decent.*
